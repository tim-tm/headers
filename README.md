# headers

This is a collection of stb-style headers I've wrote.

| **Header**  	| **Description**                                                                                  	|
|-------------	|--------------------------------------------------------------------------------------------------	|
| [hashtable.h](https://raw.githubusercontent.com/tim-tm/headers/main/include/hashtable.h) 	| A really simple hashtable implementation,<br>the file itself contains more documentation.        	|
| [logger.h](https://raw.githubusercontent.com/tim-tm/headers/main/include/logger.h)    	| A logging utility that also provides assertion macros.<br>Printing to a FILE* is also supported. 	|
| [schnur.h](https://raw.githubusercontent.com/tim-tm/headers/main/include/schnur.h)    	| String handling utility. 	|

## Examples

You can take a look at the [tests](https://github.com/tim-tm/headers/tree/main/test) if you want to know how to use each header.

Run
```sh
make test
```
in the project root to compile and run the tests.
